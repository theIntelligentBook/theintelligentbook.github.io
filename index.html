The Intelligent Book

What if you had teaching materials that understood what they teach? Robotics, electonics, and mathematics in the browser - this is a site for interactive exercises that include computer models.

In the early 2000's, the Intelligent Book project at Cambridge and MIT explored creating smart interactive online materials. My PhD was part of this project. In the years since then, AI and interactivitiy in the web browser has become a lot easier to build. So, I'm re-imagining the project for the modern world where browsers are so much more powerful and AI is everywhere.

It's early days - at the moment, this is a homepage for links to individual exercises I've created.

Exercises so far...

  • The Coding Escape - An hour of code and a little AI.
  • Diffusion - A small experiment modelling how molecules diffuse.
